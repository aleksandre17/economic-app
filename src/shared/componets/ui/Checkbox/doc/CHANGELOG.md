# Changelog

ყველა მნიშვნელოვანი ცვლილება დოკუმენტირებულია ამ ფაილში.

## [1.0.0] - 2024-12-10

### დამატებული
- ✅ საწყისი რელიზი
- ✅ Checkbox - ძირითადი კომპონენტი
- ✅ FormCheckbox - React Hook Form ინტეგრაცია
- ✅ CheckboxGroup - ჯგუფის კონტეინერი
- ✅ FormCheckboxGroup - დინამიური გენერაცია
- ✅ useCheckboxItems - დამხმარე hook
- ✅ სრული TypeScript მხარდაჭერა
- ✅ რესფონსივი დიზაინი
- ✅ Accessibility ფუნქციები
- ✅ სრული დოკუმენტაცია

### ფუნქციები
- Native HTML checkbox custom styling-ით
- ავტომატური state სინქრონიზაცია
- ვალიდაციის error-ების ჩვენება
- Helper text მხარდაჭერა
- Disabled states
- Required ინდიკატორები
- Touch-friendly დიზაინი (44px მინიმუმი)
- Mobile-first რესფონსივი დიზაინი

### დოკუმენტაცია
- სრული README
- API რეფერენსი
- გამოყენების მაგალითები
- საუკეთესო პრაქტიკები
- სწრაფი დაწყების გზამკვლევი
- ინსტალაციის ინსტრუქციები

### ტექნიკური დეტალები
- React 18+ თავსებადობა
- TypeScript 4.9+ მხარდაჭერა
- React Hook Form 7+ ინტეგრაცია
- CSS Modules სტილინგისთვის
- Responsive clamp() მნიშვნელობები
- WCAG 2.1 AA დონის accessibility
